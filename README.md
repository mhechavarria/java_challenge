# Lenio Labs Java Challenge

Small API REST for account management

## Challenge description

Develop the logic inside FeeCalculatorFactory and AccountController, taking into account that AccountController must return the fee based on the type of the account object. HINT: try to think how the factory class can return the correct implementation of the calculator interface (multiple injected instances of the same class/interface)

## DO NOT change any Spring and/or custom annotations

The solution you have to provide must be developed and built inside the above mentioned classes.

## Mocking tests are not required!

But you can include them
